<h3><a href="https://www.mlit.go.jp/tagengo-db/en/R1-02060.html">田沢湖</a></h3>
<style>
h3{
  font-size: 40px;
  color: black;
  text-align:center;
　}
</style>

<div>&nbsp;&nbsp;田沢湖は、周囲約20キロメートルのほぼ円形の湖です。
水深423.4メートル、日本一の深さを誇っています。金色の女性の像「たつこ像」は、
田沢湖の有名スポット。永遠の若さと美貌を願い、湖神になったとされる辰子姫の
ブロンズ像です。たつこ像から車で約10分の場所に辰子姫を祀る「御座石神社」の鳥居があり、
女性に嬉しいパワースポットとしても話題で、美容や縁結びに効果があるとの噂もあります。</div>

<h4><a href="https://akita-fun.jp/brandnew/29">乳頭温泉郷</a></h4>
<style>
h4{
  font-size: 40px;
  color: black;
  text-align:center;
　}
</style>

<div>&nbsp;&nbsp;外国人旅行客からも人気の高い乳頭温泉郷。七湯は独自に源泉を持ち、
その泉質は多種多様で十種類以上の源泉があります。JR新幹線「こまち」に乗って、JR田沢湖駅を目指します。
田沢湖駅から乳頭温泉郷までは羽後交通バスで40分ほどかかりますが、田沢湖や駒ヶ岳などの美しい自然を眺められるコースなので、
飽きることはありません。冬になれば里山が真っ白な雪で包まれ、いっそう風情ある景観に。
日本ならではのものを堪能したい観光客が世界中から訪れていました。</div>

<h5><a href="https://web.aiu.ac.jp/">秋田国際教養大学</a></h5>
<style>
h5{
  font-size: 40px;
  color: black;
  text-align:center;
　}
</style>

<div>&nbsp;&nbsp;国際教養大学といえば、素晴らしい建築美を誇る「24時間365日開館」の中嶋記念図書館や、
「すべての授業が英語で行われる」「1年間の海外留学が必須」という独自のカリキュラムが有名です。
国際教養大学図書館は、学生が24時間365日いつでも利用できる大学の学習拠点です。半円形状で段々状の書棚と
閲覧席は円の中央から見上げるように構成され、たくさんの「知」が用意されている事を体感させ、
本を探索する楽しさを環境的に提供しています。2014グッドデザイン賞を受賞したことがあります。
</div>
